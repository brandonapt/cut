# cut
a basic open-source implementation of https://bleed.bot (specifically it's `bleed` feature to download media)

.env file format:

```
DISCORD_TOKEN=
COBALT_URL=
BLEED_PREFIX=

JAIL_ROLE=

MODERATOR_ROLE=
ADMIN_ROLE=
```
